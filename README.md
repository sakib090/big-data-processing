# Big Data Processing with Apache Spark

Large-scale data analysis across four datasets using Apache Spark on a 
university Hadoop cluster, with outputs stored in AWS S3.

## Tasks

### Task 1 — Expedia Hotel Search Analytics
Analysed large-scale hotel search impression data. Engineered time features, 
computed click-through and conversion rates, detected position bias, and 
identified unusual search activity.

### Task 2 — Amazon Book Reviews Analysis
Processed millions of book reviews to understand rating behaviour, category 
popularity, reviewer activity patterns, and time-based trends across 18 years 
of data.

### Task 3 — NYC Yellow Taxi Graph Analytics
Built a graph of NYC taxi zones using GraphFrames. Computed PageRank, 
shortest paths, connected components, degree analysis, and community 
detection via Label Propagation.

### Task 4 — IoT Structured Streaming Pipeline
Built a real-time Spark Structured Streaming pipeline processing live smoke 
detection sensor data with watermarking, sliding windows, stateful aggregations, 
and fault-tolerant checkpointing.

## Tech Stack
- Apache Spark / PySpark
- Spark Structured Streaming
- GraphFrames
- AWS S3
- Python / Jupyter
